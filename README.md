# PowerShell 1 : Découverte du langage 

**Liste des commandes Unix et leur équivalent PowerShell** :
 - cp = Copy-Item
 - rm = Remove-Item
 - cd = Set-Location
 - mkdir = New-Item
 - man = Get-Help ou help (les deux fonctionnent)
 - history = Get-History
 - alias = Get-Alias ou Set-Alias (suivant si l'on veut afficher ou créer des alias)
 - cat = Get-Content

**Les commandes utilisées sur PowerShell sont les suivantes** :
 - Get-Help 'nom_de_l'alias' _Exemple : Get-Help cp_
 - Get-Alias 'nom_de_l'alias' _Exemple : Get-Alias history_
